# Solar-Early
Machine Learning Investigation Into Early Solar Storm Prediction Using Solar Activity Data
## Project Goal
This project investigates whether machine learning can predict solar storms before they occur.
## Research Question
How early can machine learning reliably detect warning signs of solar storms?
## Hypothesis
Machine learning models will identify patterns in solar activity that can provide earlier warnings of solar storms.
## Methods
Data:
- Solar activity measurements
- Sunspot observations
- Solar flare records
Models:
- Logistic Regression
- Random Forest
- Neural Network
